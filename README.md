Doodle Jump – Final Project

This project is a Doodle Jump–style game created as the final assignment using JavaScript and p5.js.

The player controls a character that jumps automatically and must land on platforms to climb higher without falling or hitting monsters.

⸻

Controls
	•	Move left: A / ←
	•	Move right: D / →
	•	Jump: automatic
	•	Restart: refresh page

⸻

Features
	•	Multiple platform types (normal, moving, breakable)
	•	Monsters and springs
	•	Score system based on height
	•	Start screen and game over screen
	•	Replayable gameplay
